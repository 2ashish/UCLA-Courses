Code setup using mmediting framework https://mmediting.readthedocs.io/
Assumes installation done, dataset and pretrained model downloaded

./mmediting/pruning/prune.py used to prune BasicVSR model with structured and unstructured pruning/prune
./mmediting/pruning/flops.py used to compute FLOPS for different pruned models
./mmediting/trained_ckpts contains pruned and finetune models
